# PARSER_25

Небольшой PHP-парсер для получения данных из Google Sheets через промежуточный API-ключ и вывода обработанных значений по выбранным вкладкам.

## Что делает проект

- запрашивает полный ключ для доступа к Google Apps Script;
- получает данные из Google Sheets (по вкладкам);
- извлекает значения с координатами строк/колонок;
- формирует и выводит агрегированные данные в консоль.

## Структура репозитория

- `/cron/1.php` — основной исполняемый скрипт;
- `/cron/functions/provider_google.php` — получение ключа и чтение данных из Google Sheets;
- `/cron/functions/Google_Tabl.php` — извлечение колонок/строк с координатами;
- `/cron/functions/Google_Value.php` — утилиты доступа к значениям из JSON;
- `/connect/db.php` — Singleton-подключение к MySQL через PDO.

## Требования

- PHP 8+ (рекомендуется);
- расширения PHP: `curl`, `json`, `pdo`, `pdo_mysql`;
- доступ к внешним URL, используемым в коде.

## Быстрый запуск

1. Перейдите в каталог проекта.
2. Проверьте настройки в `cron/1.php`:
   - `$Api_Key`
   - `$Provider`
   - `$Users_ID`
   - `$SheetNames`
3. Запустите:

```bash
php /home/runner/work/PARSER_25/PARSER_25/cron/1.php
```

## Примечания

- В репозитории нет настроенных команд линтинга/сборки/тестов.
- Рекомендуется вынести чувствительные значения (ключи/пароли) в переменные окружения или защищённый конфиг, а не хранить в коде.
